# Code Conventions Guidelines

Este arquivo referencia as convencoes oficiais de codigo do projeto.

- Fonte principal: `documentation/rules/code-conventions-rules.md`

## Escopo

- Naming (`PascalCase`, `camelCase`, `snake_case`).
- Organizacao de imports por camada.
- Padrao MVP para UI (`View`/`Presenter`).
- Uso de `signals` e `riverpod`.
- Imutabilidade com `final` e regras de clean code.

Em caso de divergencia, considerar `documentation/rules/code-conventions-rules.md` como fonte de verdade.
